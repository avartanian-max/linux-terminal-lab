This repo has various scripts to run 

To run any script do "./scripts/(script name).sh

Scripts include

filescript.sh - Checks if a file is present if it is return the number of lines in the file, if not create the file with the name provided
greet.sh - Returns your name and the current date
ask.sh - Asks the user for a name and returns it
process_files.sh - Creates three text files
backup.sh - Creates a backup of a given directory to another given directory
datebackup.sh - Creates a timestamped backup of a directory
networkmonitor.sh - Pings a list of hosts and reports whcih are up and down, saves to a file with timestamps
totallines.sh - Loops through all .txt files in the current directory and counts the total lines of all of them
